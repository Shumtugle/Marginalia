# Marginalia

Сайт-книга с перелистыванием страниц и аудиоверсиями.

## Как добавить новую книгу

1. В папке `books/` создаёшь новый файл `имя.txt` (латиницей, без пробелов).
2. Структура файла:

```
TITLE: Название книги
META: рассказ · 2026
COVER_FROM: #5a4530
COVER_TO: #2a1d12
COVER_ACCENT: #d4b896
AUDIO: имя.mp3
===
Первый абзац первой страницы.

Второй абзац — отделяется пустой строкой.
===
Первый абзац второй страницы.

И так далее.
```

3. `===` на отдельной строке — разделитель страниц.
4. Пустая строка внутри страницы — новый абзац.
5. `AUDIO:` можно опустить, если аудио нет.
6. Если есть mp3 — кладёшь его в папку `audio/` с тем именем, что указано в `AUDIO:`.
7. Открываешь `library.txt` и дописываешь имя файла без `.txt`.

Через минуту после коммита книга появляется на полке.

## Цвета обложки

`COVER_FROM` и `COVER_TO` — крайние цвета градиента, `COVER_ACCENT` — цвет текста на обложке. Hex-формат, можно подобрать на любом colorpicker'е.

## Технологии

- [StPageFlip](https://github.com/Nodlik/StPageFlip) — перелистывание
- Чистый HTML/CSS/JS, без сборщиков
